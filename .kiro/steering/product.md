# Product Overview

Raag Palta Trainer is an interactive web application for practicing Indian classical music patterns (paltas/alankars). It helps musicians develop muscle memory and fluency in different raags (melodic frameworks) through systematic pattern practice.

## Core Features

- Practice patterns across 10 classical thaats (parent scales)
- Customizable tempo, repetitions, and loop modes
- Multiple sound types: harmonium, flute, piano, synth
- Tanpura drone accompaniment with volume control
- Practice mode with full-screen focus
- Custom pattern creation
- Pattern categorization: Linear, Janti, Skipping, Return, Complex
- Multiple notation systems: Hindi, English, Numbers
- Adjustable Sa (tonic) note selection
- Sequential and loop playback modes

## User Experience

The app provides both a desktop layout (sidebar + main content) and responsive mobile layout. Users can select a thaat, configure playback settings, and practice patterns either individually or sequentially through all patterns in a category.
